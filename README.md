## 📖 About

Learning Machine Learning and Deep Learning from research papers alone can be overwhelming.

This project collects the **best interactive websites, GitHub repositories, visual essays, and educational creators** that explain ML concepts visually.

Instead of reading hundreds of pages of theory, you can actually **see neural networks learn**, **watch attention mechanisms compute**, and **understand diffusion models step-by-step**.

Whether you're a beginner, AI engineer, researcher, or educator, this collection helps build genuine intuition.

# 💡 Why this project?

Most ML resources explain concepts using equations.

This project focuses on **interactive understanding**.

Instead of memorizing

```
Attention(Q,K,V)
```

you can literally watch

- Queries
- Keys
- Values
- Attention Scores
- Softmax
- Output Tokens

being computed in real time.

The same philosophy applies to CNNs, GANs, Diffusion Models, optimization, and probability.
